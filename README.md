## Description

A transcription from Symfony project <a href="https://github.com/nicolas-nielsen/marketplace-api">`marketplace-api`</a> into a nest project using same db

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
